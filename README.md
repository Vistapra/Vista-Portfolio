## 🚀 Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm package manager
- Git

### Installation

1. Clone the repository

```bash
git clone https://github.com/Vistapra/Vista-Portfolio.git
```

2. Navigate to project directory

```bash
cd portfolio
```

3. Install dependencies

```bash
npm install
```

4. Start development server

```bash
npm start
```
